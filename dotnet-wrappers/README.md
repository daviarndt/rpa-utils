# .NET Wrappers

Custom .NET / VB.NET wrapper DLLs that extend Blue Prism with capabilities it
doesn't ship out of the box. Each wrapper includes its **source code** and a
**build script**, so it can be rebuilt from scratch — not just a dropped-in binary.

| Folder | What it does |
|---|---|
| [`pdfpig-reader/`](./pdfpig-reader/) | Text and word-coordinate extraction from PDFs using [PdfPig](https://github.com/UglyToad/PdfPig), exposed as Blue Prism Actions. |

See each folder's `README.md` for build and deployment instructions.
